# DOBOT TCP-IP Python API

This directory is reserved for the official DOBOT TCP-IP Python API.

This repository does NOT include any proprietary or third-party code from SHENZHEN DOBOT CORP LTD.

## Official Source

The official repository is maintained by Dobot-Arm:

https://github.com/Dobot-Arm/TCP-IP-Python-V4

## Required File

This project requires only the following file from the official repository:

    dobot_api.py

## Installation Procedure

1. Visit the official repository linked above.
2. Download or clone the repository.
3. Copy the file:

       dobot_api.py

   into this directory.

4. The resulting structure should be:

       project_root/
       └── dobot/
           ├── dobot_api.py
           ├── README.md
           └── .gitignore
